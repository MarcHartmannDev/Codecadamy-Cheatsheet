CSS Positions

A small guide to better understand Box positioning in CSS

The CSS property "position" can be used to position your HTML Element or "Box" in your document wherever you want it to be.

Position

Value Description
static Standard Value. Box is in its proposed position in the general document flow
relative Box can be moved relative from its original position in the document flow
can overlap other boxes
absolute Box is removed from the document flow and can be moved freely.
can overlap other boxes
sticky Box can be moved relative from its original position
can be fixed to a position wehn out of viewport
fixed Box is removed from the document flow and can positioned fix (even when scrolling the page)
can overlap other boxes
